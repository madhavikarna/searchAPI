Please follow the steps to run the project

1. Create a Java Project using your favorite Development editor
2. Create package called com.apple.itunes
3. Copy the iTunesSearchAPITest source code under the package
4. Add the following jars to external jars/libraries option

httpclient-4.5.3.jar
httpcore-4.4.6.jar
java-json.jar
junit4.3.1.jar
commons-logging-1.2.jar

5. Run the application, it will execute 10 jUnit test cases, details of 
test cases are 

1. Search without passing any parameters
2. Search by using term parameter
3. Search by term with Empty parameter
4. Search by Country parameter 
5. Search by invalid Country parameter
6. Search by Media
7. Search by Invalid Media
8. Search by Limit parameter
9. Search by Default limit parameter
10. Search by Invalid Limit parameter
